# C / C++ Basics

## What is C?
C is a procedural, low-level programming language used for system programming.

## What is C++?
C++ is an extension of C that supports object-oriented programming.

## Variables
A variable stores data.
```c
int x = 10;
```

## Data Types
- int
- float
- double
- char

## Input / Output
```c
printf("Hello");
scanf("%d", &x);
```

## Difference: C vs C++
- C uses printf/scanf
- C++ uses cin/cout
